# SOP_GPT_TO_OS.md

How to move a conversation or thread from ChatGPT into the symbolic OS.

## Step-by-Step:

1. Export or copy text from the GPT session.
2. Paste into a `.md` file in `XP-PR_00-00_experiments-live/`.
3. Create a UID tag (e.g. XP-PR_01-03) at the top of the file.
4. Update `UID_MAP.md` inside the XP-PR folder.
5. If it's ready for myth, fork to `NR-TH/` or `SB-ML/`.

Use `thread_parser.py` (future) to automate this process.
