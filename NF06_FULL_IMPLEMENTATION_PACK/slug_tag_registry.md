# Slug & Tag Registry

## Tag Bank
- ASSET: Media or object used in another composition  
- SCRIPT: Code or automation logic  
- SOURCE: Raw file or original draft  
- EXPORT: Final rendered or delivered file  
- DOC: Human-readable instructions or overview  
- PATCH: A bug fix, revision, or small intervention  
- MANIFEST: An index or container of related files  
- MODULE: Self-contained, reusable unit  
- SEED: Generative origin object  
- TEMPLATE: Reusable scaffold for future objects  
- NODE: Connective or architectural unit  
- RITUAL: System-bound symbolic file
